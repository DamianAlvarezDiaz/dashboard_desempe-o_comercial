# dashboard_desempe-o_comercial
Proyecto de Bootcamp TripleTen, donde la dirección de Andes Retail Group necesita un dashboard interactivo que permita entender el desempeño comercial de 2024–2025.  Aquí se transforman datos transaccionales dispersos en información visual, clara y accionable que permita responder preguntas sobre ventas, rentabilidad y comportamiento de clientes
